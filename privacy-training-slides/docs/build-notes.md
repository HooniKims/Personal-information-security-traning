# 제작 메모

## 슬라이드 구성

원본 `개인정보보호 연수.md`의 5분 연수 구조를 발표용 웹 슬라이드로 재구성했습니다. 대본은 제외하되, 화면만 보고도 5분 발표 흐름이 유지되도록 핵심 기준, 현장 예시, APM 절차를 함께 배치했습니다.

1. 오프닝
2. 학교 개인정보의 위험성
3. 법적 책임
4. 수집 고지 4요소
5. 과잉 수집 점검
6. 제3자 제공과 업무 위탁 비교
7. 파기 원칙
8. 디가우징과 저장매체 폐기
9. APM 1~3번 절차
10. APM 4~6번 절차
11. APM 영구삭제
12. APM 예외 처리
13. 현장 실수 3가지
14. 사고 대응
15. 3대 골든 액션
16. 아웃트로

## 디자인 기준

- 폰트: Pretendard Variable 로컬 파일
- 자간: `-0.02em`
- 줄간격: `1.33`
- 화면비: 16:9
- 조작 방식: 웹 발표와 전체화면 발표 모두 지원
- 효과음: 장면 전환마다 클릭, 페이지 넘김, 알림, 셔터 계열 효과음 사용
- 생성 이미지: `$imagegen`으로 제작한 페이지별 배경과 중심 오브젝트를 사용
- 움직임 표현: 배경 확대/이동, 스캔 라인, 포인트 마커, APM 컷 테두리 펄스, 디가우징 자기장 링, 방패 호흡, 파쇄 오브젝트 흔들림
- 텍스트 연출: 장면마다 `surge`, `alarm`, `check`, `split`, `erase`, `scan`, `lock`, `final` 효과를 다르게 적용
- 파비콘: 자물쇠 방패 오브젝트 기반 `favicon.png`, `favicon.ico`

## PDF 컷 사용 위치

- `apm_steps_1_2.png`: 삭제 방법 안내, APM 아이콘 클릭, 암호 생성 또는 입력
- `apm_steps_3_4.png`: 위험도 막대 더블클릭, 파일 리스트 체크박스 선택
- `apm_steps_5_6.png`: 암호화 권장, 위험/경고/주의 0건 확인
- `apm_exception_full.png`: 예외 처리 확인 및 예외 해제 절차

## 생성 이미지 사용 위치

- `bg_opening_flow.png`: 오프닝
- `generated_incident.png`: 위험성
- `generated_actions.png`: 3대 골든 액션
- `generated_degauss.png`: 파기, 디가우징, 저장매체 폐기
- `bg_*.png`: 법적 책임, 수집, 과잉 수집, 제공/위탁, APM, 현장 실수, 사고 대응, 아웃트로 등 페이지별 전용 배경
- `element_*.png`: 슬라이드별 전용 중심 오브젝트. 16개 슬라이드가 서로 다른 움직이는 개체를 사용
